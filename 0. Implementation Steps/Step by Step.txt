***********************BASIC JWT IMPLEMENTATION W/O PASSPORT*********************************

Register JWT in AuthModule and Use JwtService to sign Payload in login signup
Implement AuthGuard service to protect endpoint by having the request have a token for validation
Enable Global Use of Authentication
Configure the provider in AuthModue {provide: APP_GUARD,useClass: AuthGuard}
        This makes all endpoint use the custom AuthGuard class we implemented
Now since every endpoint use Authguad the route which is used to assign token is also locked behind AuthGuard
Hence we implement @Public Decorator to make routes public which bypasses the AuthGuard in AuthGuard class and use @Public() with the endpoint which doesnt use AuthGuard

Note: Here we made Custom Authguard with no stratergy to implement guards in endpoints and also made all endpoints secure and are making the endpoints which do not need Guards as Public
In Passport Integration we'll use default AuthGuard by Passport and use our custom jwt stratergy to validate user and attach the payload to the request
